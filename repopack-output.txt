<h5>💡 <strong>그때, 무슨 일이 있었을까?</strong></h5>
<br>

<p>윤동주는 1945년, 일본의 후쿠오카 형무소에서 짧은 생을 마감했지만, 그의 시는 여전히 많은 사람들에게 감동을 주고 있습니다. 일제강점기의 억압과 불의에 저항하며 자신을 희생한 윤동주는 그의 시 ‘하늘과 바람과 별과 시’로 대표되는 저항 정신을 남겼습니다. 영화는 그와 그의 사촌 송몽규의 삶을 조명하며, 그 시대의 청년들이 겪어야 했던 억압과 내적 갈등을 섬세하게 그립니다.</p>
<hr>
<h5><strong>🎬 당시의 사회적 분위기</strong></h5>
<br>

<p>1930-1940년대의 한국은 일제의 압박 속에서 독립운동이 점차 조직화되던 시기였습니다. 많은 청년들이 교육을 위해 일본으로 유학을 떠났지만, 그곳에서도 차별과 억압을 겪었습니다. 윤동주와 송몽규 같은 인물들은 문학과 지식으로 저항하고자 했고, 그들의 활동은 일본의 감시와 탄압을 피해 나아갔습니다. 이 시기는 한국 문학사에서 중요한 전환점이 되었으며, 많은 저항 문인들이 탄생한 시기이기도 합니다.</p>

            ### 1950년 9월 15일: **인천상륙작전 - UN군의 전략적 반격**

            1950년 9월, 한반도의 대부분이 북한군의 통제 아래에 있었다. 북한군은 남한의 거의 모든 주요 도시를 점령했고, 한국군과 유엔군은 낙동강 방어선까지 후퇴한 상태였다. 이 절망적인 상황에서
            맥아더 장군이 이끄는 유엔군은 인천상륙작전(코드명: Operation Chromite)이라는 대규모 반격 작전을 계획했다. 인천은 북한군의 방어가 취약한 지점이었지만, 지형적으로 상륙이 매우
            어려운 지역이었다. 그럼에도 불구하고, 맥아더는 이 작전의 성공을 통해 전세를 역전할 수 있다고 판단했다. 1950년 9월 15일, 작전이 성공하면서 서울을 탈환하고 전쟁의 흐름을 바꾸는
            결정적인 계기가 되었다.

            '인천상륙작전'은 이 역사적 사건을 배경으로, 작전의 성공을 위해 활약한 정보부원들과 맥아더 장군의 결단을 중심으로 이야기를 풀어간다. 영화는 군사적 전략과 첩보 활동의 중요성을 강조하며,
            위험한 상황에서도 작전을 성공시키기 위해 목숨을 걸고 싸운 사람들의 이야기를 담고 있다. 첩보원들이 수행한 비밀 작전과 상륙작전의 긴박한 전개를 통해 전쟁의 전략적 측면을 부각시킨다. 이 영화는
            전쟁의 거대 작전이 어떻게 전쟁의 흐름을 바꾸었는지, 그리고 이를 성공으로 이끌기 위해 어떤 희생이 있었는지를 잘 보여준다.

            ## 6. 현대적 함의: 대한민국 정치사에서 12.12의 중요성

            12.12 사태는 대한민국 군부의 정치 개입과 정권 장악을 상징하는 중대한 사건으로, 현대 정치사에서 군부와 정치의 관계를 재조명하게 한 계기다. 이 사건을 통해 군부는 정치권에서 결정적인
            영향력을 행사하며 권력을 쥐었고, 이는 이후 1980년대 군사 정권으로 이어졌다. 12.12 사태는 민주주의 후퇴와 권위주의 정권의 부상을 알린 사건으로, 군의 정치적 중립성에 대한 논의와
            제도적 변화를 촉발시켰다.

            오늘날 12.12 사태는 정치적 군사 행위의 위험성과 함께 민주주의와 권위주의 사이에서 권력의 균형을 어떻게 유지해야 하는지에 대한 중요한 교훈을 제공한다. 이 사건은 군부의 권력 남용을
            경계하게 하고, 이후 군의 정치적 중립성을 확립하는 데 중요한 영향을 미쳤다. 12.12는 대한민국이 민주적 체제로 가는 과정에서 피할 수 없었던 군사적 개입의 역사를 이해하는 데 필수적인
            사건이다.

            ## 7. **사건 이후의 정치적 변화**: 군부 독재로의 전환

            전두환과 신군부는 12.12 사태를 통해 군사력을 동원하여 권력을 장악한 후, 군사 정권을 수립하며 대한민국의 정치 체제를 강하게 흔들었다. 이 과정에서 민주적 절차는 무시되었고, 권위주의적인
            정치 체제가 1980년대에 들어서면서 강화되었다. 5.18 광주 민주화 운동은 이 권위주의에 맞선 대표적인 시민 저항이었다. 광주에서 시민들은 민주주의를 요구했으나, 군부는 강경하게 이를
            진압하며 국가의 권력을 더욱 공고히 했다.

            군부는 광주 민주화 운동을 무력으로 진압함으로써 정권에 대한 도전을 억압했고, 이는 이후 전두환이 이끄는 권위주의 체제를 공고히 하는 중요한 계기가 되었다. 이 시기에는 언론과 정치 활동이

              광주에서 직접 촬영한 자료입니다. 힌츠페터는 당시 군부의 철저한 봉쇄 속에서도 목숨을 걸고 광주의 진실을 기록했으며, 그가 촬영한 영상은 군부의 잔혹한 학살을 국제사회에 처음으로 알린 중요한

    </div>

    <!-- 프롬프트 입력 영역 -->
    <footer class="footer">
      <input type="text" v-model="userInput" placeholder="메시지" @keyup.enter="sendMessage" class="message-input" :disabled="isLoading" />

<script>

import axios from 'axios'
const BASE_URL = 'http://127.0.0.1:5000/'

export default {
  components: {
  },
  data() {
    return {
      userInput: "", // 새로운 메시지를 저장
      chatHistory: [], // 채팅 메시지 목록
      isLoading: false, // 로딩 상태
    };
  },
  methods: {
    async sendMessage() {
      if (this.userInput.trim() !== "") {
        // 메시지가 비어 있지 않을 때만 전송
        this.chatHistory.push({ message: this.userInput, isUser: true }); // 보낸 메시지는 사용자 메시지로 분류
        this.isLoading = true; // 로딩 상태 시작
        // this.userInput = ""; // 입력창 비활성화

        try {
          // URL 로 요청을 보냄
          const res = await axios.post(BASE_URL, {
            message: this.userInput,
          });
          // 받은 응답메시지를 chatHistory에 추가함
          this.chatHistory.push({ isUser: false, message: res.data.llm });
        } catch (error) {
          console.error("Error sending message:", error);
        } finally {
          this.isLoading = false; // 로딩 상태 끝
          this.userInput = ''
        }
      }
    },
  },
};
</script>
